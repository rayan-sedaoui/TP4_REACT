#  TP 4 React : My Super To-Do List

Bienvenue sur le dépôt de mon quatrième TP en React ! Ce projet est une application web interactive de gestion de tâches (To-Do List), développée pour consolider mes compétences sur la gestion de l'état en React.

##  Fonctionnalités

Cette application propose les fonctionnalités suivantes :
- **Ajout de tâches** : Saisie d'une nouvelle mission via un formulaire contrôlé.
- **Validation (Exercice complémentaire)** : Affichage d'un message d'erreur personnalisé si l'utilisateur tente d'ajouter une tâche vide.
- **Gestion de l'état (Cocher/Décocher)** : Possibilité de marquer une tâche comme terminée (le texte se barre et le design de la ligne change dynamiquement).
- **Suppression** : Bouton dédié pour retirer définitivement une tâche de la liste.
- **Compteur dynamique** : Affichage en temps réel du ratio de tâches terminées (ex: 1 / 3).
- **Design UI/UX soigné** : Utilisation de styles React inline avec des ombres portées, des bords arrondis et des couleurs modernes.

##  Technologies utilisées

- **React.js** (Composants fonctionnels)
- **Hooks** : Utilisation intensive de `useState` pour gérer la liste des tâches, la saisie utilisateur et les messages d'erreur.
- **CSS-in-JS** : Stylisation conditionnelle directement dans les composants.
